## Features

- **Dual shadow system** — Inner shine + soft drop shadow for subtle 3D depth
- **Dynamic colors** — Pass any color, auto-calculates text contrast
- **Multiple sizes** — sm, default, lg, icon variants
- **Zero dependencies** — Pure React + Tailwind

## Interactions

| Trigger | Action |
|---------|--------|
| Click | Scale down feedback (0.97) |
| Hover | Cursor pointer, subtle state change |
| Focus | Ring outline for accessibility |
| Disabled | 50% opacity, no pointer events |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | - | Background color (hex, rgb, named) |
| `size` | `"sm" \| "default" \| "lg" \| "icon"` | `"default"` | Size variant |
| `disabled` | `boolean` | `false` | Disables the button |
| `className` | `string` | - | Additional classes |

## Usage

```tsx
import { BadgeButton } from "@/components/ui/badge-button";

// With color
<BadgeButton color="#FDAC16">Primary</BadgeButton>

// Different sizes
<BadgeButton size="sm" color="#1e3a5f">Small</BadgeButton>
<BadgeButton size="lg" color="#0e3a0e">Large</BadgeButton>

// Neutral (no color)
<BadgeButton>Neutral</BadgeButton>
```
