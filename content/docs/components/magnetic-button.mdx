## Features

- **Magnetic attraction** — Button smoothly follows cursor position
- **GSAP powered** — High-performance, 60fps animations
- **Touch support** — Works seamlessly on mobile devices
- **Flair effects** — Optional pulse animation on hover

## Interactions

| Trigger | Action |
|---------|--------|
| Hover | Button magnetically follows cursor within range |
| Mouse Leave | Button smoothly returns to center |
| Click | Scale feedback + onClick handler fires |
| Touch | Same magnetic behavior on mobile |

## Usage

```tsx
import { MagneticButton } from "@/components/ui/magnetic-button";

<MagneticButton onClick={() => console.log("Clicked!")}>
  Hover me
</MagneticButton>

// With custom color
<MagneticButton hoverVariant="custom" customColor="#ff6b00">
  Custom Color
</MagneticButton>
```
