## Features

- **Magnetic attraction** — Button follows cursor with smooth animation
- **GSAP powered** — High-performance animations
- **Touch support** — Works on mobile devices
- **Flair effects** — Optional pulse animation on hover
- **Multiple variants** — light, dark, and custom colors

## Interactions

| Trigger | Action |
|---------|--------|
| Hover | Button magnetically follows cursor |
| Mouse Leave | Button returns to center position |
| Click | Triggers onClick handler with scale feedback |
| Touch | Same magnetic behavior on mobile |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Button content |
| `onClick` | `() => void` | - | Click handler |
| `hoverVariant` | `"light" \| "dark" \| "custom"` | `"light"` | Hover effect style |
| `customColor` | `string` | - | Custom color (hex/rgb/hsl) |
| `className` | `string` | - | Additional CSS classes |
