## Features

- **Shared layout transitions** — Smooth background animation between chips
- **Spring physics** — Natural, bouncy animations
- **Multiple variants** — default, outline, ghost, pill
- **Type-safe** — TypeScript generics for filter types
- **Flexible** — Works as links or buttons

## Interactions

| Trigger | Action |
|---------|--------|
| Click Chip | Active indicator animates to clicked chip |
| Hover | Chip background lightens |
| Focus | Keyboard accessible with Tab |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `filters` | `Array<{value, label}>` | - | Array of filter options |
| `activeFilter` | `T` | - | Currently active filter value |
| `onFilterChange` | `(value: T) => void` | - | Callback when filter changes |
| `variant` | `"default" \| "outline" \| "ghost" \| "pill"` | `"pill"` | Visual style variant |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Chip size |

## Variants

- **default** — Filled background style with rounded corners
- **outline** — Border style with transparent background
- **ghost** — Minimal style, background on hover only
- **pill** — Fully rounded pill shape
