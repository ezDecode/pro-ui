## Features

- **Intent-aware delays** — Shows tooltip faster when user shows clear intent
- **Keyboard pinning** — Pin tooltip with Cmd/Shift key
- **Smooth animations** — Powered by Framer Motion
- **Micro-reveal effects** — Anime.js stagger animations
- **Accessible** — Full keyboard navigation support

## Interactions

| Trigger | Action |
|---------|--------|
| Hover | Tooltip appears after intent delay |
| Quick Hover | Shorter delay for experienced users |
| Cmd/Shift + Hover | Pins tooltip open |
| Mouse Leave | Tooltip fades out |
| Tab Focus | Keyboard accessible |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `TooltipItem[]` | - | Array of tooltip items |
| `enablePinning` | `boolean` | `true` | Enable Cmd/Shift pinning |
| `className` | `string` | - | Additional CSS classes |

### TooltipItem

| Prop | Type | Description |
|------|------|-------------|
| `icon` | `ReactNode` | Icon element to display |
| `label` | `string` | Tooltip text |
| `shortcut` | `ReactNode[]` | Optional keyboard shortcut keys |
| `hasBadge` | `boolean` | Show notification badge |

## Dependencies

- `framer-motion` - Animation transitions
- `animejs` - Stagger micro-reveal effects
- `clsx` - Utility for conditional classes
