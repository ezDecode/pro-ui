## Features

- **Intent-aware delays** — Shows tooltip faster when user shows clear intent
- **Keyboard pinning** — Pin tooltip with Cmd/Shift key
- **Smooth animations** — Powered by Framer Motion
- **Micro-reveal effects** — Anime.js stagger animations
- **Accessible** — Full keyboard navigation support

## Usage

```tsx
import { AdaptiveTooltip, AdaptiveTooltipItem } from "@/components/ui/adaptive-tooltip";

export function MyComponent() {
  return (
    <AdaptiveTooltip>
      <AdaptiveTooltipItem
        trigger={<button>Hover me</button>}
        content="Tooltip content"
      />
    </AdaptiveTooltip>
  );
}
```

## Props

### AdaptiveTooltip

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Tooltip items to render |
| `className` | `string` | - | Additional CSS classes |

### AdaptiveTooltipItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `React.ReactNode` | - | Element that triggers the tooltip |
| `content` | `React.ReactNode` | - | Tooltip content to display |
| `className` | `string` | - | Additional CSS classes |

## Accessibility

- Supports keyboard navigation with Tab key
- Pin tooltip with Cmd (Mac) or Shift (Windows) key
- ARIA labels for screen readers
- Focus management
