## Features

- **Scroll-locked animation** — Text reveals as you scroll
- **Word-by-word reveal** — Each word animates individually
- **Glassmorphic pills** — Subtle background effects on words
- **Responsive** — Optimized for mobile and desktop
- **Highlight support** — Emphasize specific words with custom colors
- **Custom font** — Uses PolySans for distinctive typography

## Interactions

| Trigger | Action |
|---------|--------|
| Scroll Down | Words reveal progressively with glassmorphic pill effect |
| Scroll Up | Animation reverses, words fade back |
| Reach End | All text fully visible |

## Animation Phases

Each word goes through three phases:

1. **Emergence** (0-15%) — Pill box fades in with scale
2. **Focus** (15-60%) — Pill reaches full opacity
3. **Reveal** (60-100%) — Text fades in, pill fades out

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `phrase` | `string` | - | The text content to reveal (required) |
| `title` | `string` | - | Optional heading above the text |
| `highlightWords` | `string[]` | `[]` | Words to highlight with primary color |
| `primaryColor` | `string` | `"#ff6b00"` | Color for highlighted words (hex) |
| `scrollContainerRef` | `RefObject` | - | Custom scroll container |

## Dependencies

- `framer-motion` - Animation library for scroll-based transitions
