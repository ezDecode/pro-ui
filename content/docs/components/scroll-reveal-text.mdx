## Features

- **Scroll-locked animation** — Text reveals as you scroll
- **Word-by-word reveal** — Each word animates individually
- **Glassmorphic pills** — Subtle blur effects on emerging words
- **Highlight support** — Emphasize specific words with custom colors

## Interactions

| Trigger | Action |
|---------|--------|
| Scroll Down | Words reveal progressively with pill effect |
| Scroll Up | Animation reverses, words fade back |
| Reach End | All text fully visible |

## Animation Phases

Each word transitions through three phases:

1. **Emergence** (0-15%) — Pill fades in with scale
2. **Focus** (15-60%) — Pill at full opacity  
3. **Reveal** (60-100%) — Text fades in, pill dissolves

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `phrase` | `string` | - | Text content to reveal (required) |
| `title` | `string` | - | Optional heading above text |
| `highlightWords` | `string[]` | `[]` | Words to highlight |
| `primaryColor` | `string` | `"#ff6b00"` | Highlight color |
| `scrollContainerRef` | `RefObject` | - | Custom scroll container |

## Usage

```tsx
import { ScrollRevealText } from "@/components/ui/scroll-reveal-text";

<ScrollRevealText
  phrase="Your text content goes here and reveals word by word as you scroll."
  highlightWords={["reveals", "scroll"]}
  primaryColor="#FDAC16"
/>
```
